# 草神桌面宠物（带对话记忆）

一个可爱的桌面宠物 —— 纳西妲，她会跟随你的鼠标移动，随机发出声音，并且你可以按空格键与她对话（基于通义千问 API，支持多轮记忆）。

## ✨ 功能特点
- **鼠标跟随**：角色会平滑跟随鼠标移动。
- **待机模式**：右键点击角色可切换待机/跟随模式；在待机模式下右键空白区域可退出待机。
- **随机语音**：待机时会随机播放闲置语音（需将音频文件放在 `voice/` 文件夹）。
- **智能对话**：按空格键唤出输入框，输入问题后纳西妲会通过气泡和语音回答，且能记住之前的对话内容。
- **背景音乐**：点击页面任意位置即可播放背景音乐。
